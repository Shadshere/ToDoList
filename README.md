# ToDoList
A simple To-Do list for Windows Desktop App using C# and .NET Framework
WATCH HERE live demo : https://youtube.com/shorts/mJHle85A86Y 

## 🚀 Features
- Add and delete tasks
- Edit tasks
- Mark tasks [DONE]
- Minimal and clean UI

## 🖥️ Technologies
- C#
- .NET Framework
- Windows Forms
- Visual Studio
